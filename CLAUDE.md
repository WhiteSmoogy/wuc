# CLAUDE.md

This file provides guidance to Claude Code when working with code in this repository.

## Project Overview

**Wuc** is a Unity-to-Agent bridge. It exposes Unity Editor capabilities (scene inspection, object manipulation, C# script execution, log access) to AI agents via a local HTTP server.

### Architecture

```
Claude Code / AI Agent
       │
       │  python wuc.py <action>
       ▼
  wuc.py  (HTTP client, project root)
       │
       │  discovers ~/.wuc/instances, verifies /identity
       │  HTTP  127.0.0.1:<dynamic-port>
       ▼
  WucServer.cs  [InitializeOnLoad] — Assets/Editor/Wuc/
       │
       │  main-thread dispatch via EditorApplication.update
       ▼
  CSharpScriptRunner.cs  (Roslyn scripting + Temp/wuc.log persistence)
       │
       ▼
  Unity Engine / Editor API
```

## Key Files

| File | Role |
|------|------|
| `Assets/Editor/Wuc/WucServer.cs` | HTTP server, route dispatch, main-thread queue |
| `Assets/Editor/Wuc/WucSettings.cs` | Project settings (port range and projectId override) |
| `Assets/Editor/Wuc/CSharpScriptRunner.cs` | Roslyn C# executor, append-only `Temp/wuc.log` persistence |
| `Assets/Editor/Wuc/Plugins/` | Roslyn DLLs + System.Text.Json bundled |

## HTTP API (dynamic port)

Unity registers each running editor at `~/.wuc/instances/<instanceId>.json`.
The client resolves by `projectId` and confirms with `GET /identity`.

### GET /identity
Returns `{ projectId, projectPath, instanceId, pid, port, startedAtUtc }`.

### POST /execute
```json
{ "code": "<C# expression or statements>", "scriptPath": "Label.csx", "timeoutMs": 30000 }
```
Response: `{ success, returnValue, output, logs, executionTimeMs, error }`

### GET /logs?count=N
Returns last N Unity log entries (default 100) from `Temp/wuc.log`.
Each entry: `{ timestamp, timestampUtc, type, message, stackTrace }`

### POST /logs/clear
Clears all entries from `Temp/wuc.log`. Returns `{ ok: true, removedCount }`.
Use this before asking an agent to decide whether a fresh compile emitted errors.

### POST /logs/clear-before
```json
{ "before": "2026-03-08T12:34:56.789Z" }
```
Removes entries earlier than `before`. Prefer passing the `timestampUtc` value returned by `/logs`.

## C# Scripting (Roslyn)

Scripts run via `Microsoft.CodeAnalysis.CSharp.Scripting` on the **Unity main thread**.

**Auto-imported namespaces:** `System`, `System.Linq`, `System.Collections.Generic`, `UnityEngine`, `UnityEditor`

**All loaded assemblies** are referenced automatically — no manual `using` needed for Unity types.

**ScriptGlobals helpers available in every script:**
- `print(value)` — appends to `output` field in response
- `log(value)` — calls `Debug.Log`

**Return values:** the last expression value is returned as `returnValue` (`.ToString()`).

## Development Notes

- Server starts automatically via `[InitializeOnLoad]`; restarts on assembly reload.
- All Unity API calls are dispatched to the main thread via `ConcurrentQueue` + `EditorApplication.update`.
- `System.Text.Json` is used for serialization (bundled in `Plugins/`, not Newtonsoft).
- Do not edit `Wuc.sln` — it is auto-generated by Unity.
